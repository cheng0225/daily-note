# 08 - 历史快照（v1.8）

每次**重置之前**自动存档；App 底部 **「历史」** Tab 可展开只读对比。

## 功能

- 底部导航：**今日** | **历史**
- 列表行：时间 + 来源 + `完成数/总数`（如 `2/10`）
- 点击行下方展开重置前的任务（只读，可多行同时展开）
- 暂不设保留上限，历史一直累积

## 何时记快照

`TaskRepository.resetAllTasks(source)` 在清空打钩**之前**调用 `saveSnapshot()`：

| source | 场景 |
|--------|------|
| `manual` | App「立即重置」 |
| `widget` | 桌面便签「重置」 |
| `scheduled` | 定时闹钟 |
| `catchup` | 打开 App 补重置 |

## 数据表

- `reset_snapshots`：时间、完成数、总数、来源
- `reset_snapshot_items`：快照时的任务文案与完成状态（副本）

Room 数据库 version **2**，`MIGRATION_1_2` 保留原有任务数据。
