# 06 - 设置页

设置页代码在 `SettingsActivity.kt` + `activity_settings.xml`。

## 入口

主界面右上角 **「设置」** 按钮 → 打开 `SettingsActivity`。

## 桌面便签透明度

| 滑块值 | 效果 |
|--------|------|
| 0% | 背景完全透明（默认） |
| 50% | 半透明白色底 |
| 100% | 不透明白色底 |

### 数据存哪

```kotlin
// TaskRepository.kt
KEY_WIDGET_BG_OPACITY  // SharedPreferences，0～100
```

### 怎么应用到小组件

`TaskWidgetProvider` 更新时用 `RemoteViews.setInt` 设置根布局背景色：

```kotlin
views.setInt(R.id.widget_root, "setBackgroundColor", repository.getWidgetBackgroundColor())
```

`getWidgetBackgroundColor()` 把 0～100 转成 ARGB 白色：`Color.argb(alpha, 255, 255, 255)`。

拖动滑块会**即时保存**并调用 `TaskWidgetProvider.refreshAll()` 刷新桌面。

## 小组件标题与名称

- v1.1 起：便签上 **不再显示「今日任务」标题**，只保留任务列表
- 桌面小组件**下方名称**由系统 Launcher 绘制，应用只能把 `android:label` 设为空白尝试隐藏；vivo 等机型可能仍会显示，属系统限制
