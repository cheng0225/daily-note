# 01 - 项目总览

## 这个 App 做什么

「每日便签」= **桌面透明便签** + **每日任务打钩** + **定时重置**。

```
用户
 ├── 打开 App → 管理任务（增删改、打钩、设重置时间）
 └── 桌面小组件 → 看任务、点一下打钩（不用开 App）
```

## 技术栈一览

| 技术 | 用途 | 在本项目中的文件 |
|------|------|------------------|
| Kotlin | 开发语言 | 所有 `.kt` 文件 |
| Room | 本地数据库 | `data/` 目录 |
| SharedPreferences | 存重置时间、上次重置日期 | `TaskRepository.kt` |
| RecyclerView | App 内任务列表 | `MainActivity` + `TaskAdapter` |
| App Widget | 桌面小组件 | `widget/` 目录 |
| AlarmManager | 每天定点重置 | `ResetScheduler.kt` |
| BroadcastReceiver | 接收闹钟、开机广播 | `receiver/` 目录 |
| Coroutines | 异步数据库操作 | `lifecycleScope.launch` 等 |

## 目录结构

```
daily-note/
├── app/src/main/
│   ├── AndroidManifest.xml    # 注册 Activity、小组件、广播
│   ├── java/.../dailytasks/   # Kotlin 源码
│   └── res/
│       ├── layout/            # 界面布局 XML
│       ├── values/            # 颜色、字符串、主题
│       └── xml/               # 小组件配置
├── docs/                      # 本学习文档
├── releases/                  # 可安装的 APK
└── gradlew.bat                # 命令行打包
```

## 应用启动流程

```
1. 系统启动 App
      ↓
2. DailyTasksApp.onCreate()
      → 注册每日重置闹钟（ResetScheduler.schedule）
      ↓
3. MainActivity.onCreate()
      → 检查是否需要补重置（ensureDailyReset）
      → 订阅任务列表（observeTasks → 自动刷新 UI）
```

## 数据存在哪

| 数据 | 存储位置 | 说明 |
|------|----------|------|
| 任务列表（文案、顺序、是否完成） | SQLite（Room） | 文件 `daily_tasks.db` |
| 每日重置时间（时、分） | SharedPreferences | 默认 08:00 |
| 上次重置日期 | SharedPreferences | 用于判断今天是否已重置 |
