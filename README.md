# Android 工作区

个人 Android 项目集合。

## 目录说明

```
Android/
├── vibe-coding/     # 需求与规划文档（非代码）
└── daily-note/      # 每日便签 — Android 工程
```

| 目录 | 类型 | 说明 |
|------|------|------|
| [vibe-coding](vibe-coding/) | 文档 | 产品需求、技术方案、开发路线 |
| [daily-note](daily-note/) | 代码 | 「每日便签」App 源码 |
| [scripts/publish-github.ps1](scripts/publish-github.ps1) | 脚本 | 一键推送 GitHub 并发布 Release |

## 快速开始

1. 阅读 [vibe-coding/docs/REQUIREMENTS.md](vibe-coding/docs/REQUIREMENTS.md) 了解需求
2. 用 Android Studio 打开 `daily-note/` 目录运行项目
