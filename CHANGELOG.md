# Changelog

## [1.0.0] - 2026-07-03

首个正式版本。

### 功能

- App 内添加 / 编辑 / 删除每日任务（最多 15 条）
- App 与桌面小组件同步打钩
- 桌面小组件透明便签样式
- 每日定时重置（默认 08:00，可修改，改时间类似改闹钟）
- 立即重置、开机 / 打开 App 补重置
- 本地 Room 存储，无需联网

### 文档

- `vibe-coding/` 产品需求文档
- `daily-note/docs/` 源码学习文档（01～05）
